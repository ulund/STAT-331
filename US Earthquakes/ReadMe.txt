US Earthquake Data:

Downloaded from USGS Earthquake Catalog at 
https://earthquake.usgs.gov/earthquakes/search/

Search Criteria were
* Minimum 3 magnitude
* 00:00:00 of current year to 00:00:00 of next year
* Conterminous US
* Event Type = Earthquakes
* Format = CSV
* Order By = Oldest first

Conterminous US is defined by rectangular region using these coordinates:
    [24.6, 50]  Latitude
    [-125, -65] Longitude
